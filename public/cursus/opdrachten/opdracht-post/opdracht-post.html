Opdracht post: deel 1

  • maak een html-formulier volgens de regels van de kunst
    • de action is de naam van de huidige pagina
    • gebruik een POST-method
    • In dit formulier zitten twee inputvelden:
      • username
      • paswoord (van het type paswoord dus!)
      • vergeet niet om een name-attribuut aan deze velden toe te kennen
      • gebruik een label (werk dus met een id op de input-velden)
    • Het resultaat ziet er ongeveer als volgt uit:

      Inloggen

  • In de php- script tag (bovenaan) doe je het volgende:
    • maak een variabele $password met een string die je zelf kiest als paswoord (bv. 'azerty')
    • maak een variabele $username met een string die je zelf kiest als gebruikersnaam (bv. 'stijn')
    • controleer of het formulier gesubmit is
      • werk dus met de isset() functie
      • zorg ervoor dat er geen Undefined index zich voordoet. Als dit voorkomt wil dit zeggen dat je geen of niet nauwkeurig met de isset() hebt gewerkt.
    • Als het formulier gesubmit is, mag je controleren of de gebruikersnaam en het paswoord die de gebruiker heeft opgegeven, gelijk zijn aan de waarde die je zelf in de variabele hebt geplaatst (bv. azerty/stijn)
      • Als deze niet overeenkomen geef je een foutboodschap: 'Er ging iets mis bij het inloggen, probeer opnieuw'
      • Als deze overeenkomen toon je een boodschap: 'Welkom'
      • Plaats deze boodschap in een variabele $message die je onderaan in de HTML toont.
