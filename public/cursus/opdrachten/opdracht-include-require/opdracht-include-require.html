  • De mappen/bestandenstructuur ziet er als volgt uit:
    • view
      • header-partial.html
      • body-partial.html
      • footer-partial.html
    • index.php

  • Stap 1: HTML pagina aanmaken en versnijden

    • Maak eerst een HTML-pagina aan met een geldige HTML opmaak.
    • In de body zit minstens een paragraaf met wat tekst.
    • Je mag dit zo uitgebreid maken als je zelf wil (header met nav, footer, …)
    • Versnijd deze dan in de volgende delen:

      • header-partial.html

        • Dit bestand bevat een geldige HTML opmaak met de vereiste tags tot minstens de opening body tag (html, head, title, body) en de header met navigatie van je pagina als die in je design voorkomen.

      • body-partial.html

        • Hierin komt alles wat tussen de body-container staat, afhankelijk van hoe je html-pagina er uit ziet.

      • footer-partial.html

        • Hierin komt het einde van je pagina met eventuele footer, afhankelijk van hoe je html-pagina er uit ziet.

  • Stap 2: HTML weer samenstellen

  • Dit gebeurt in de index.php
  • Maak in dit bestand een variabele $artikels aan. Deze bevat een multidimensionale array met de volgende structuur:
    • [0] =>
      • ['title'] => Titel van artikel 1
      • ['text'] => Tekst van artikel 1
      • ['tags'] =>
        • [0] => tag 1 van artikel 1
    • [1] =>
      • ['title'] => Titel van artikel 2
      • ['text'] => Tekst van artikel 2
      • ['tags'] =>
        • [0] => tag 1 van artikel 2
        • [1] => tag 2 van artikel 2
    • [2] =>
      • ['title'] => Titel van artikel 3
      • ['text'] => Tekst van artikel 3
      • ['tags'] =>
        • [0] => tag 1 van artikel 3
        • [1] => tag 2 van artikel 3
        • [2] => tag 3 van artikel 3
  • Voer de nodige includes van de header, body en footer uit.
  • Zorg er nu voor dat in de body de $artikels array wordt overlopen en de artikels worden uitgelezen
  • Als dit gelukt is, controleer je via de W3 HTML-validator of je html valid is. Als dit niet het geval is, maak je de nodige aanpassingen.
