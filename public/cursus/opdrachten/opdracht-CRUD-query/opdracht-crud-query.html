Opdracht CRUD query: deel 1

  • Maak een connectie met de lokale MySQL server en selecteer de database bieren (deze vind je in /public/cursus/bieren.sql)
  • Zorg ervoor dat wanneer er niet kan geconnecteerd worden met de database, er een custom foutboodschap verschijnt, inclusief de specifieke fout.
  • Voer de volgende query uit: selecteer alles uit de tabel bieren die gelinkt is aan de tabel brouwers waarvan de naam van het bier begint met du en de brouwernaam een a bevat.
  • Als deze query niet onmiddellijk lukt, probeer hem dan eerst in de PHPMyAdmin uit. Als hij daar werkt, kan je hem copy/pasten.
  • Bouw een <table> met de gevonden resultaten.
    • Werk met een <thead> waarin je de kolomnamen zet. De eerste kolomnaam is voorbestemd voor het nummer van het resultaat. Misschien kan de static je hierbij helpen?
    • <tfoot> mag leeg blijven
    • In de <tbody> komen alle gevonden resultaten.
      • Even rijen moeten een aparte klasse krijgen. Deze rij moet via CSS gestijld worden en een andere achtergrondkleur krijgen.
      • Als dit lukt kan je eens onderzoeken hoe je dit puur met CSS kan oplossen, zonder dat je bv. een klasse odd op elke oneven rij moet toepassen.
    • Het resultaat ziet er ongeveer als volgt uit:

Opdracht CRUD query: deel 2

  • Maak een connectie met de lokale MySQL server en selecteer de database bieren
  • Zorg ervoor dat wanneer er niet kan geconnecteerd worden met de database, er een custom foutboodschap verschijnt, inclusief de specifieke fout.
  • Voer de volgende query uit: selecteer de brouwernummer en de brouwernaam
  • Bouw een formulier op volgens de regels van de kunst via een get-methode en de action verwijst naar dezelfde pagina.
    • In dit formulier bevindt zich een select. Deze select bevat alle name van elke brouwerij.
      • De value van elke brouwernaam is gelijk aan het brouwernummer
      • Wanneer er op de submit wordt gedrukt moet er een tabel verschijnen met alle bieren van deze brouwerij
        • Er moet een nieuwe query uitgevoerd worden. De bieren moeten geselecteerd worden op basis van het brouwernummer
        • De resultaten moeten in een <table> weergegeven worden en werk met <thead>, <tfoot> en <tbody>
        • Voor de naam van elk bier staat de nummer van het hoeveelste bier dit is. Misschien kan de static je hierbij helpen?
        • De oneven rijen krijgen een klasse odd. Deze klasse heeft een andere achtergrondkleur
  • Het resultaat ziet er ongeveer als volgt uit
    Aantal naam
    1 Big Chouffe (=Chouffe )
    2 Chouffe
    3 Chouffe-bok 6666

    9 Vieille Salme
  • Zorg ervoor dat de brouwernaam waarvan de bieren getoond worden, geselecteerd is nadat de query is uitgevoerd.
