Opdracht CRUD insert: deel 1

  • Bouw de volgende pagina na volgens de regels van de kunst:

    Voeg een brouwer toe

  • Het name attribute in elk <input> element krijgt als value de overeenstemmende kolomnaam zoals die in de database onder de tabel brouwers staat
  • Het name attribute bij de submit-knop krijgt submit als value
  • Wanneer er op toevoegen wordt geklikt (werk hiervoor met de value van het name attribute attribute van de submit-knop) moet de informatie toegevoegd worden aan de tabel 'brouwers'.
    • Maak een connectie met de database en selecteer de database bieren.
    • Zorg ervoor dat wanneer er niet kan geconnecteerd worden met de database, er een custom foutboodschap verschijnt, inclusief de specifieke fout.
    • Voer de insert-query uit
    • Als deze query niet onmiddellijk lukt, echo hem dan eerst. Zie je niet onmiddellijk een fout, kopieer hem dan in de PHPMyAdmin console en kijk naar de foutmeldingen.
    • Wanneer de query gelukt is, moet er een boodschap getoond worden: 'Brouwerij succesvol toegevoegd. Het unieke nummer van deze brouwerij is XXX', waarbij de XXX de primary key is van de toegevoegde datarij. Zoek eens naar insert id voor de PDO classe
    • Wanneer dit niet lukt moet de volgende boodschap getoond worden: 'Er ging iets mis met het toevoegen. Probeer opnieuw.'
